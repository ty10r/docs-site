MUnit Utilities

MUnit provides a set of utilities for you to write your tests in a portable way and mock external system connections from your tests.

You can start a Database and an FTP server in your local environment and open a Run and Wait Scope to configure your automated tests to start just after the threads opened by your production code have finished processing.
